# 密码生成器

Rpass 的密码生成器帮助创建强壮、唯一的密码。

## 使用生成器

![密码生成器界面](/img/password_generator.png)

如上图所示，Rpass 密码生成器界面包含以下主要功能：

1. **密码显示区**：生成的密码会以不同颜色显示不同类型的字符（如字母、数字、特殊字符），并显示密码强度。
2. **刷新按钮**：点击可重新生成一个新密码。
3. **密码长度调节**：通过滑块调整生成密码的长度。
4. **包含字符类型**：可选择生成密码时包含哪些字符类型，包括：
    - 小写字母（abc）
    - 大写字母（ABC）
    - 数字（01）
    - 特殊字符（!@）
    - 自定义字符集
5. **确认按钮**：设置好参数后，点击右下角的蓝色按钮即可生成符合要求的密码。

通过灵活配置上述选项，用户可以快速生成安全且符合需求的密码。
