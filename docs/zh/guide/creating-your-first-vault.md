# 创建第一个密码库

密码库是 Rpass 中存储所有密码数据的容器。本指南将引导你创建你的第一个密码库。

## 什么是密码库？

密码库是一个加密的数据库，包含你的所有登录凭据、笔记和其他敏感信息。每个密码库都由主密码保护。

## 创建步骤

1. 启动 Rpass 并登录。
2. 在主界面，点击 "新建密码库" 或类似按钮。
3. 输入密码库名称，例如 "个人密码"。
4. 选择存储位置：
   - 本地文件：存储在你的电脑上。
   - 云同步：如果配置了 WebDAV 等。
5. 设置加密选项（默认即可，使用 AES-256）。
6. 点击 "创建"。

## 添加第一个条目

创建密码库后：

1. 点击 "添加新条目"。
2. 输入网站/应用名称。
3. 输入用户名。
4. 输入密码，或使用密码生成器。
5. 保存。

你的第一个密码库现在可以使用了！

## 提示

- 定期备份你的密码库文件。
- 不要分享主密码。
- 如果需要多个密码库，可以创建不同的库用于不同目的。
