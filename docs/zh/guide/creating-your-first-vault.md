# 创建第一个密码库

密码库是 Rpass 中存储所有密码数据的容器。本指南将引导你创建你的第一个密码库。



## 初始化主密码

> **注意** 主密码是唯一能解密密码库的密钥，如果遗忘将无法恢复数据

<img height="573" alt="init_main_password" src="/img/init_main_password.png" />

1. 默认开启，需要密码
2. 密钥文件，可选的，但如果禁止了密码，则必须添加，点击可选择现有key文件
3. 生成一个随机的key文件并选择
> 如果同时设置了密码和密钥文件，在后续登录都需要用上
