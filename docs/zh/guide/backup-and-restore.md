# 备份与恢复

备份你的密码库是确保数据安全的关键。本指南介绍如何备份和恢复 Rpass 数据。

## 备份密码库

1. 在 Rpass 中，点击 "文件" > "备份密码库"。
2. 选择备份位置（外部硬盘、云存储等）。
3. 选择是否加密备份文件（推荐）。
4. 点击 "备份"。

### 自动备份

- 设置定期备份：在设置中启用自动备份。
- 选择频率（每日、每周）。
- 指定备份目录。

## 恢复密码库

1. 如果需要恢复，安装 Rpass。
2. 点击 "文件" > "恢复密码库"。
3. 选择备份文件。
4. 输入主密码（如果加密）。
5. 恢复到新位置或覆盖现有。

## 备份最佳实践

- 存储在多个位置（本地 + 云）。
- 测试恢复过程。
- 不要只依赖云备份。
- 定期验证备份完整性。

## 紧急情况

如果设备丢失，备份文件可以让你在新设备上恢复所有数据。
