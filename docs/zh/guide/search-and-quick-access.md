# 搜索与快速访问

Rpass 提供强大的搜索功能，帮助你快速找到密码。

## 搜索规则

> **[字段名:]**["]关键句["]

- 中括号包裹的是可选的
- 支持多个条件搜索，按空格分割，如果关键句有空格则需要用双引号包裹。

| 字段名称      | 匹配方式 | 是否包含在默认搜索中 |
| :--:         | :--:   | :--:              |
| title(t)     | 包含    | 是                |
| url[1-5]     | 包含    | 是                |
| user(u)      | 包含    | 是                |
| email(e)     | 包含    | 是                |
| note(n)      | 包含    | 是                |
| password(p)  | 包含    | 是                |
| OTPAuth(otp) | 包含    | 是                |
| tag          | 完全匹配 | 是                |
| group(g)     | 完全匹配 | 否                |
| 自定义字段    | 包含    | 否                |


> 括号内表示缩写，中括号内表示复数（url1, url2）。


## 例子

1. 搜索用户名包含**小明**，备注了**到此一游，小明**。

```
u:小明 n:"到此一游, 小明"
```

2. 搜索邮箱分组下的用户**小明**。
```
g:邮箱 u:小明
```
