# 主密码设置与密码策略

主密码是访问你的 Rpass 密码库的关键。本指南解释如何设置和维护主密码，以及密码策略。

## 设置主密码

1. 在创建账户或密码库时设置。
2. 要求：
   - 至少 12 个字符
   - 包含大写字母、小写字母、数字和特殊字符
   - 不要使用常见单词或个人信息

## 密码策略

Rpass 可以强制执行密码策略，确保生成的密码强壮。

### 配置密码策略

1. 进入设置 > 密码生成器。
2. 设置最小长度（默认 12）。
3. 启用字符类型：大写、小写、数字、符号。
4. 禁用相似字符或重复。

### 应用策略

- 生成新密码时自动应用。
- 可以为不同类型条目设置不同策略。

## 安全提示

- 定期更改主密码。
- 使用密码管理器记住主密码。
- 启用双因素认证如果可用。
- 不要在公共设备上输入主密码。

## 忘记主密码

由于数据是本地加密的，忘记主密码意味着数据丢失。始终备份密码库文件。
