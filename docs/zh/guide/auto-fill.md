# 自动填充

自动填充功能让登录变得快速便捷。本指南介绍如何设置和使用自动填充。

## 启用自动填充

1. 在 Rpass 设置中，启用 "自动填充"。
2. 安装浏览器扩展（如果适用）。
3. 授予必要权限。

## 如何使用

1. 访问登录页面。
2. Rpass 会检测并提示填充。
3. 点击填充或使用快捷键。

## 浏览器支持

- Chrome
- Firefox
- Safari
- Edge

## 配置

- 设置填充延迟。
- 排除某些网站。
- 自定义匹配规则。

## 安全注意

- 只在可信网站填充。
- 定期检查填充的数据。
- 如果填充失败，手动输入。
