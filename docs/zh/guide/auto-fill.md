# 自动填充

自动填充功能让登录变得快速、便捷。本指南介绍如何设置和使用自动填充。

## Android

前往设置＞安全＞更多设置＞启用自动填充服务

在系统设置中选择 **Rpass Autofill Service**

> 开启手动选择数据项（默认关闭），当在填充匹配失败时弹出选择提示。


## Windows

![Windows 自动填充](/img/windows_autofill.gif)

想必你也注意到了，是根据填充序列规则，通过模拟键盘进行填充。


> 规则支持自定义，细节可以查看[自定义填充序列](/guide/custom-fill-sequences)。



## Macos

工作流程和 Windows 一样。

但在这之前需要设置辅助权限。

前往系统设置：隐私与安全性＞辅助功能

如果看到 Rpass，请确保启用状态。
如果没有，请尝试添加并启用。
