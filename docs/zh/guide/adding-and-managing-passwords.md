# 添加与管理密码

学习如何在 Rpass 中添加新密码条目并管理现有条目。

## 编辑账号条目

Rpass 的“编辑账号”页面用于修改已经保存的密码条目。页面分为多个区域，帮助你组织登录信息、自动填充规则、扩展字段以及附加备注。

### 项目信息

- **分组**：用于将账号条目归类到不同的文件夹或分组中，例如“默认”、“工作”或“个人”。
- **标题**：条目的主名称，用于快速识别该账号，例如“公司邮箱”或“淘宝账户”。

### 账号信息

该区域包含主要的账号凭证：

- **账号**：登录账号名或用户名。
- **邮箱**：与该账号关联的电子邮箱地址。
- **密码**：该条目的当前密码，可直接查看或编辑。
- **一次性密码（OTP）**：如果该账号启用了二次验证，可在这里保存一次性密码或二维码链接信息。

### 自动填充信息

自动填充部分用于配置浏览器或应用中的自动登录规则：

- **自动填充应用**：选择要启用自动填充的应用或浏览器环境。
- **自动序列**：在桌面端，按照一定规则，通过模拟键盘的方式填充密码。
- **域名**：指定该账号适用的网站域名。
- **添加域名**：支持为同一个账号设置多个域名，适用于同一账号可登录多个子站点或域名的情况。

### 自定义字段

如果默认字段无法满足需求，可以通过“自定义字段”添加额外条目：

- 点击“添加字段”来创建新的自定义输入项。
- 例如：安全问题答案、PIN 码、备用账号、API Key 等。

### 附加信息

附加信息区域可以记录更多补充内容：

- **描述**：自由文本说明，用于记录账号用途、注意事项或特殊登录方式。
- **标签**：为账号添加标签，便于后续搜索与筛选，例如“常用”、“团队”、“支付”等。
- **附件**：上传与账号相关的文件、截图或文档备份。
- **过期时间**：如果密码有失效周期，可设置到期日期，方便及时更新密码。

> 你可以为每个账号条目填写完整信息，确保在需要时快速找到对应登录凭证，并让自动填充逻辑准确地匹配目标站点。
