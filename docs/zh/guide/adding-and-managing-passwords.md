# 添加与管理密码

学习如何在 Rpass 中添加新密码条目并管理现有条目。

## 添加新密码

1. 打开密码库。
2. 点击 "添加" 或 "+" 按钮。
3. 填写信息：
    - 标题：网站或应用名称
    - 用户名
    - 密码：手动输入或生成
    - URL
    - 笔记：额外信息
4. 保存。

## 编辑密码

1. 选择条目。
2. 点击 "编辑"。
3. 修改字段。
4. 保存。

## 删除密码

1. 选择条目。
2. 点击 "删除"。
3. 确认（可能移到回收站）。

## 管理技巧

- 使用标签分组条目。
- 搜索快速找到。
- 克隆条目创建相似。
- 自定义字段添加更多信息。

## 批量操作

- 选择多个条目进行批量删除或移动。
