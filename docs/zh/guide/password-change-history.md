# 密码变更历史

每一次保存密码条目后，Rpass 都会自动生成一条变更历史记录。你可以在密码浏览页面的菜单中，通过“历史记录”功能查看每个条目的历史变更详情。

## 功能说明

- **自动记录**：每次对密码条目进行保存（如新增、修改、删除字段等）时，系统会自动记录当前条目的完整快照。
- **历史查看**：在密码浏览页，点击条目右侧菜单，选择“历史记录”即可查看该条目的所有历史版本。


## 使用场景

1. **误操作恢复**：如果不小心修改或删除了重要信息，可以通过历史记录找回。
2. **变更追踪**：方便了解账号信息的变更过程，提升安全性和可追溯性。

## 注意事项

- 历史记录仅针对每次保存操作生成，未保存的编辑不会被记录。
- 历史记录占用空间较小，但建议定期清理无用条目，保持数据整洁。

> 通过历史记录功能，你可以轻松管理和追溯所有密码条目的变更过程，保障数据安全。
