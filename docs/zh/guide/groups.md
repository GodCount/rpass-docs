# 分组

使用分组来组织你的密码条目，使管理更高效。

## 编辑分组界面

![编辑分组界面](/img/edit_groups.png)

如上图所示，编辑分组界面包含以下主要部分：

1. **项目信息**：
    - 标题：输入或编辑分组的名称。
    - 显示设置：继承显示设置（默认），包含在主页列表中，排除在主页列表外。
    - 搜索设置：继承搜索设置（默认），包含在搜索中，排除在搜索外。
2. **附加信息**：
    - 描述：为分组添加补充说明。

## 管理分组项目

支持对分组内项目进行批量操作。

1. 移动多个项目到其他分组。
2. 删除分组内多个项目。
