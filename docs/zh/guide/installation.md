# 安装卸载

安装卸载 Rpass。


## 下载 Rpass

1. 访问 [GitHub 发布页面](https://github.com/GodCount/rpass-flutter/releases/latest) 。
2. 下载适合你操作系统的安装包。
    - Android：Rpass-vx.x.x-arch.apk
        > 如果不清楚设备架构，可以下载**arm64-v8a**（大部分都是这个架构）
    - Windows：Rpass_windows_vx.x.x.zip
    - macOS：Rpass-vx.x.x.dmg

## 安装步骤

### Windows
1. 安装
    -  windows 是便携版压缩包形式，解压完双击 rpass.exe 运行即可
    -  默认配置在 C:\Users\{YourName}\AppData\Roaming\com.godcount\Rpass
2. 卸载
   - 直接删除整个文件夹即可
   - **注意** 配置文件不会被删除，配置保存了密码库的加密信息。卸载后建议删除。

### macOS
1. 安装
    -  打开下载的 .dmg 文件。
    -  将 Rpass 图标拖拽到 Applications 文件夹。
    -  从 Applications 启动 Rpass。
2. 卸载
    -  把 Rpass 移到回收站即可
    -  **注意** 配置文件不会被删除，配置保存了密码库的加密信息。卸载后建议删除。
