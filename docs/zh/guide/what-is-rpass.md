# 什么是 Rpass？

Rpass 是一款开源的密码管理器，旨在提供安全、便捷的密码存储和管理解决方案。它能帮助你轻松管理各种账户的登录信息，生成强密码，并确保你的数据安全无忧。

## 主要特性

- **安全存储**：使用先进的加密技术保护你的密码数据。
- **自动填充**：在浏览器和应用中自动填充登录信息。
- **密码生成器**：创建强壮、唯一的密码。
- **跨设备同步**：在多个设备间同步你的密码库。
- **开源免费**：完全开源，免费使用。

## 为什么选择 Rpass？

Rpass 专注于隐私和安全，确保你的数据只属于你自己。没有云存储依赖，你可以选择本地存储或自托管同步。
