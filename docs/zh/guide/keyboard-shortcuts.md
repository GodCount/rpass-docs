# 快捷键

使用快捷键提高 Rpass 使用效率。

## 支持的快捷键

| 标题 | 描述 | 默认快捷键 |
| :---: | :--- | :---: |
| 打开Rpass | 显示或隐藏窗口 | Alt + Shift + A |
| 锁定 | 锁定密码库并回到验证界面 | Alt + Shift + L |
| 自动填充 | 根据填充序列，填充选中密码项 | Alt + Shift + Enter |
| 填充用户名 | - | Alt + Shift + 1 |
| 填充邮箱 | - | Alt + Shift + 2 |
| 填充密码 | - | Alt + Shift + 3 |
| 填充 OTP | - | Alt + Shift + 4 |

> **注意**自动填充是通过模拟键盘输入的，会和快捷键冲突，按下后需要快速释放。
