# 导入已有密码

如果你已经有其他密码管理器或浏览器存储的密码，可以轻松导入到 Rpass 中。

## 支持的导入格式

- CSV 文件（从 LastPass、1Password 等导出）
- JSON 文件
- 浏览器导出（Chrome, Firefox 等）

## 导入步骤

1. 在 Rpass 中打开密码库。
2. 点击 "导入" 或 "工具" > "导入密码"。
3. 选择导入来源：
   - 从文件导入
   - 从浏览器导入
4. 选择文件或浏览器。
5. 映射字段：确保用户名、密码、URL 等字段正确映射。
6. 点击 "导入"。

## 从浏览器导入

### Chrome

1. 打开 Chrome 设置 > 密码。
2. 点击 "导出密码"。
3. 保存 CSV 文件。
4. 在 Rpass 中导入该 CSV。

### Firefox

类似步骤。

## 注意事项

- 导入后，原文件可以删除。
- 检查导入的数据是否正确。
- 如果有重复，Rpass 会提示合并或跳过。
- 导入不会覆盖现有条目，除非指定。

完成导入后，你的所有密码都在 Rpass 中了。
