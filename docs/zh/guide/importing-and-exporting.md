# 导入和导出

Rpass 支持多种密码导入和导出格式，方便你从其他浏览器或密码管理器迁移数据，并将密码备份到通用文件格式。

## 支持的导入格式

- Chrome 浏览器导出的 CSV
- Firefox 浏览器导出的 CSV

## 支持的导出格式

- KDBX（KeePass 数据库格式）
- XML
- CSV

## 导入步骤

1. 在 Rpass 中打开“设置”。
2. 找到“备份” > “导入”。
3. 选择导入来源：
   - Chrome CSV
   - Firefox CSV
4. 选择对应导出的 CSV 文件。
5. 点击“导入”。

## 从浏览器导入

### Chrome

1. 打开 Chrome 设置 > 密码。
2. 点击“导出密码”。
3. 保存生成的 CSV 文件。
4. 在 Rpass 中选择 Chrome CSV 并导入。

### Firefox

1. 打开 Firefox 设置 > 隐私与安全 > 登录信息与密码。
2. 点击“导出登录信息”。
3. 保存生成的 CSV 文件。
4. 在 Rpass 中选择 Firefox CSV 并导入。

## 导出步骤

1. 在 Rpass 中打开“设置”。
2. 找到“备份” > “导出”。
3. 选择目标格式：
   - KDBX
   - XML
   - CSV
4. 选择保存位置并确认。

## 注意事项

- 导入前请先检查 CSV 文件是否包含正确的字段和账号信息。
- 导入不会自动覆盖现有条目，请根据提示选择是否合并或跳过重复项。
- 导出时请妥善保管生成的文件，特别是 XML 和 CSV 文件，因为它们包含敏感登录数据。
- 完成导入/导出后，可按需删除原始文件以保护隐私。
